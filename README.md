# 多語言支持系統使用說明

## 概述
這是一個使用 Python 和 SQLite 架構的多語言支持系統，支援日文、英文和中文的即時切換。

## 系統需求
- Python 3.9 或更高版本
- 標準庫 (sqlite3, tkinter, threading, json, os, sys 等)

## 如何運行應用程式

### 方法一：使用批處理文件 (Windows)
1. 確保您的系統已安裝 Python 3.9 或更高版本
2. 點擊 `start.bat` 文件
3. 應用程式將在新的視窗中啟動

### 方法二：使用命令列
1. 打開命令提示字元或終端
2. 導航到項目目錄
3. 執行以下命令：
```
python main.py
```

## 功能說明

### 語言切換
- 在主界面中的下拉選單選擇您想要的語言 (日文/英文/中文)
- 界面將立即切換到所選語言

### 資源管理
- 點擊「管理」按鈕可打開資源管理介面
- 可在此新增、編輯、刪除或多語言翻譯資源
- 支援搜尋和過濾功能

### 測試功能
- 點擊「測試」按鈕可測試翻譯鍵的翻譯結果
- 可輸入翻譯鍵和默認值進行測試

### 匯入功能
- 點擊「匯入」按鈕可從 JSON 文件匯入翻譯資源
- JSON 檔案格式應為鍵值對結構

## 預設翻譯資源
系統預設包含了常用界面元素的翻譯，如：
- 標題文字
- 按鈕標籤
- 導航項目
- 通用術語

## 數據庫
- 所有翻譯資源儲存在 `language_resources.db` SQLite 資料庫中
- 包含三個主要表格：
  - `language_resources`: 翻譯資源
  - `language_settings`: 語言設置
  - `language_packs`: 語言包

## 技術架構
- 後端：Python + SQLite
- 前端：tkinter GUI
- 國際化：自定義翻譯管理系統
- 數據持久化：SQLite 數據庫

## 注意事項
1. 確保所有 Python 檔案都在同一目錄中
2. 應用程式會自動創建所需的數據庫和表
3. 關閉應用程式前請確保所有更改已儲存
4. 如遇到問題，可在終端中查看錯誤訊息

## 組織結構
- `main.py`: 主入口點
- `language_manager.py`: 語言資源管理系統
- `gui_app.py`: 圖形用戶界面
- `language_resources.db`: 數據庫文件（運行後自動創建）
- `start.bat`: Windows 啟動腳本

## 疑難排解
如果應用程式未能啟動：
1. 確認已安裝 Python 3.9 或更高版本
2. 檢查所有必要的 Python 檔案是否存在
3. 在命令提示字元中運行 `python main.py` 並查看錯誤訊息
4. 確認沒有其他應用程式正在使用 `language_resources.db` 檔案
